## FMTNA Experiment: Failure-Mode-Typed Neural Abduction

### What Was Implemented
A full neuro-symbolic reasoning pipeline (FMTNA) was implemented in `method.py` and evaluated on 60 examples (30 RuleTaker depth-0/1 + 30 FOLIO) against an undifferentiated LLM baseline.

### Pipeline Architecture
1. **Seed Extraction**: LLM (google/gemini-3.1-flash-lite) extracts structured facts and Horn-clause rules from natural language context into a Python-based Prolog Knowledge Base.
2. **Custom Prolog Interpreter**: Python SLD resolution engine with failure-type classification:
   - **Type-A** (predicate mismatch): goal functor absent from KB entirely → 9 occurrences
   - **Type-B** (missing ground atom): functor present as facts-only but no matching ground term → 16 occurrences  
   - **Type-C** (absent rule head): no clause with matching head for multi-hop → 8 occurrences
3. **Typed LLM Dispatch**: Each failure type triggers a precisely constrained LLM prompt (synonym bridging for A, document grounding check for B, world-knowledge Horn clause proposal for C).
4. **Undifferentiated Baseline**: Same KB extraction and resolution, but proof failures get a generic 'what fact or rule would help?' prompt with no failure-type classification.
5. **LLM-Judge Hallucination Annotation**: Claude Haiku 4.5 annotates each non-resolution proof step for hallucination presence and confidence.

### Key Results (60 examples, $0.043 total cost)
- **FMTNA accuracy**: 41.7% vs **baseline accuracy**: 36.7% (+5.0 percentage points)
- **FMTNA hallucination rate**: 41.7% vs **baseline**: 43.3% (−1.7pp reduction)
- **Failure type distribution**: Type-A=9, Type-B=16, Type-C=8 (typed dispatch covers all three)
- **Grounding-hallucination Pearson r = −0.695** (strong negative correlation: higher grounding ratio → lower hallucination probability)
- **Grounding ratio**: mean=0.472, std=0.461 across examples
- **Cost**: $0.0007/example, $0.043 total (well within $10 budget)

- **FOLIO** (Han et al., EMNLP 2024): expert-written FOL reasoning with True/False/Uncertain labels
