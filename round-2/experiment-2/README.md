## Layman Summary

Tests whether classifying logic-proof failures into three types and routing each to a specialized AI prompt improves reasoning accuracy over a generic approach, on 300 examples from RuleTaker and FOLIO benchmarks.

## Full Summary

Implements Failure-Mode-Typed Neural Abduction (FMTNA) across RuleTaker (depths 0,1,3,5; 200 examples stratified 50/depth) and FOLIO (100 examples) benchmarks, with full baseline comparison.

**Method**: Pure-Python backward-chaining Prolog interpreter (MiniProlog) classifies proof failures into three types: Type-A (functor vocabulary mismatch—query predicate not in KB), Type-B (missing ground atom—predicate exists but specific instance absent), Type-C (missing rule head—no applicable Horn clause). Each failure type dispatches to a specialized LLM prompt via OpenRouter (google/gemini-3.1-flash-lite). BM25 retrieval grounds Type-B resolutions to source spans. Negation-as-failure handles 'does not' queries under closed-world assumption.

**Baseline**: Undifferentiated abduction—single generic 'what is missing?' LLM prompt plus direct LLM answer, without typed dispatch.

**Key Results**:
- RuleTaker FMTNA accuracy: 73.5% vs baseline 65.5% (+8% overall, +22% at depth-1)
- Depth analysis: Type-B failures increase with reasoning depth (48.8% at depth-0 → 69.4% at depth-3), confirming that deeper reasoning chains require more ground-atom resolution
- Type-A hypothesis confirmed (chi-square p<0.001): FOLIO vocabulary-diverse examples show 100% Type-A failure rate vs 39.2% in RuleTaker, validating that diverse-vocabulary documents produce more predicate-name mismatches
- FOLIO FMTNA accuracy: 57.0% vs baseline 61.0% (FOLIO harder due to rate-limit failures and complex FOL)
- Total API cost: $0.14 / $10 budget (677 LLM calls)

**Output format**: exp_gen_sol_out schema with predict_fmtna and predict_baseline fields per example, plus metadata_fmtna_failure_types, metadata_fmtna_grounding_ratio, metadata_fmtna_proof_steps per example. Comprehensive results_summary in metadata including per-depth tables, chi-square test, and baseline comparison.
